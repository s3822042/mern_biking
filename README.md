## Clone the project

```bash
git clone https://github.com/s3822042/mern_biking.git
cd mern_biking
```

## Getting Started

First, run the client server:

```bash
cd client
npm run start
# or
yarn start
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

Second, run the backend server:

```bash
cd server
npm run server
# or
yarn server
```
